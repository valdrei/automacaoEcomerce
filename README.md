# automacaoEcomerce

Motivação:

Foi optado pela utilização do Selenium WebDriver para a automação de testes do e-comerce devido a sua praticidade de configuração e a boa compatibilidade com os principais navegadores do mercado. Neste projeto foi utilizado Selenium com Java, porém é possível utilizar o Selenium com outras linguagens de programação.


Instruções de uso:

É possível executar os testes de duas maneiras:

1ª Executando arquivo .jar;
2ª Executando os testes dentro de uma IDE;


1º Executando arquivo .jar.

Para executar o arquivo .jar é necessário que o Java esteja instalado na máquina aonde o mesmo for ser executado. Com Java instalado vá até a pasta do projeto (desafioCIandT) e faça os passos abaixo:
	- Abra a pasta driversNavegadores copie os dois arquivos .exe e cole dentro do C:;
	- Volte até a pasta do projeto e de um double click no arquivo desafioCIandT_testes.jar;
	- Pronto! Os testes irão iniciar automaticamente e serão executados em sequência.
	
2º Executando os testes dentro de uma IDE.

Para executar os testes a partir de uma IDE basta copiar a pasta do projeto (desafioCIandT) para dentro do workspace  da IDE desejada. E fazer os seguintes passos abaixo:
	- Abra a pasta driversNavegadores copie os dois arquivos .exe e cole dentro do C:;
	- Volte a pasta do projeto (dentro da IDE);
	- Selecione a pasta (src/test/resources), dentro desta pasta está contido os testes e os pageObjetcs;
	- Selecione um teste e de play.
	- Faça o passo anterior para os outros dois testes restantes.
